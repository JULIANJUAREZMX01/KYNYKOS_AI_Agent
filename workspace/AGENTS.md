- **Doble Flujo (PC ↔ Móvil)**:

## 📜 Unbreakable Rules (Modo Bypass)

- **Modo Restringido**: Por defecto, KYNIKOS valida cada comando contra la política de `SAFE_PATHS`.
- **Modo Bypass**: Si el comandante utiliza la **Palabra Clave Maestra**, todas las validaciones de seguridad de directorios y comandos restringidos se congelan temporalmente para permitir ajustes quirúrgicos en el hardware o sistema de archivos.
