# SKILL: ROI Analyst (KYNIKOS Engine)

Este framework de razonamiento permite a KYNYKOS evaluar la viabilidad técnica y financiera de cualquier propuesta antes de sugerirla a Julian.

## Framework de Evaluación

Cuando KYNYKOS propone una mejora, debe pasar por este filtro:

1. **Costo de Oportunidad**: ¿Cuánto tiempo manual se ahorra? (Meta SAC: Ahorro de 3h a 15min).
2. **Robustez Técnica**: ¿Es una solución "Offline-first"? ¿Usa motores de alto rendimiento (WASM/Rust)?
3. **Escalabilidad**: ¿Se puede aplicar a otros CEDIS o proyectos de Julian?
4. **Impacto en el Negocio**: ¿Reduce errores de inventario o tiempos de transporte?

## Ejemplo de Aplicación

- **Propuesta**: Script de validación de barcodes.
- **ROI**: Evita multas de proveedores por datos erróneos ($$$) y reduce el tiempo de recibo en 40%.
- **Veredicto**: Ejecución prioritaria.

"Si no genera valor, es basura digital."
